## 💼 오늘 완료한 작업 (2025.12.07)

### 1. 백테스트 기간 선택 옵션 확장 ✅
**기존 문제**: 백테스트 기간이 최근 1년으로 제한되어, 전략의 장기적 성과(금융위기, 코로나 등 특수 상황)를 검증하는 데 한계가 있었음.

**개선 내용**:
- **옵션 다양화**: 1Y, 3Y, 5Y, 7Y, 10Y, Max(20년) 옵션 추가
- **데이터 처리 최적화**: 대용량 데이터 조회 시의 성능 이슈 해결
- **자동 기간 계산**: 종료일 기준 역산하여 정확한 시작일 자동 설정

**기대 효과**: 다양한 시장 사이클(상승장, 하락장, 횡보장)에서의 전략 견고성(Robustness) 검증 가능.

### 2. 차트 시각화 및 UI 개선 ✅
**개선 내용**:
- **X축 날짜 포맷 변경**: 장기 데이터 조회 시 가독성을 위해 `yyyy-MM-dd` 형식 적용
- **Tooltip 개선**: 차트 호버 시 날짜 및 데이터 포맷 통일
- **RoadmapDialog 버그 수정**: 로드맵 데이터 구조 오류 수정 및 렌더링 안정화

### 3. 시스템 빌드 안정성 확보 ✅
- **TypeScript 컴파일 오류 해결**: `RoadmapDialog.tsx` 등 핵심 컴포넌트의 문법 오류 수정
- **빌드 테스트 통과**: `npm run build` 성공 (Exit Code 0) 확인
